## Обзор

**Название:** «Аудит автоматизации за 20 минут: найдите, где ваш бизнес теряет деньги прямо сейчас»

**Формат:** Чеклист + мини-таблица расчёта (Google Doc / Notion / PDF)

**Стадия покупателя:** Осведомлённость → Рассмотрение. Для тех, кто чувствует боль, но ещё не знает с чего начать.

**Почему этот формат:**
- Чеклист потребляется за 10–20 минут — низкий барьер
- Содержит расчётную часть: владелец видит конкретные цифры своих потерь
- Создаёт «разрыв осознанности»: человек понимает масштаб проблемы, но не знает как её решить → естественный переход к офферу
- Не требует дизайна и сложной разработки — можно запустить быстро

**Механизм доставки:**
- Ссылка на Google Doc с правом просматривать (без авторизации)
- Дублировать в Notion как шаблон для дублирования
- PDF-версия для скачивания

**Фрейминг opt-in в LinkedIn:**
> «Сделал чеклист — 7 вопросов, которые помогут найти, где ваш бизнес теряет деньги на рутине прямо сейчас. С мини-расчётом в рублях. Бесплатно. Пишите в личку "чеклист" — пришлю.»

Три пути: сделать самому / поставить задачу команде / обратиться к эксперту. Без навязчивого питча, но с чётким следующим шагом.

## Часть 2. Считаем потери

**Заполните таблицу для каждого «Да»:**

| Процесс | Часов в неделю | × Недель в месяц | × Стоимость часа (₽) | = Потери в месяц (₽) |
|---------|----------------|-----------------|---------------------|---------------------|
| Обработка заявок | | × 4 | × | |
| Ввод данных | | × 4 | × | |
| Документы | | × 4 | × | |
| Отчёты | | × 4 | × | |
| Типовые ответы | | × 4 | × | |
| **Итого** | | | | **= ₽/мес** |

*Если не знаете стоимость часа: зарплата сотрудника / 22 рабочих дня / 8 часов = ₽/час*

---

**Ваш результат:**

- **0–1 «Да»:** Хорошо выстроенные процессы. Возможно, есть точечные улучшения.
- **2–3 «Да»:** Есть конкретные области для оптимизации. Автоматизация одного-двух процессов быстро окупится.
- **4–5 «Да»:** Значительные потери. Автоматизация приоритетных процессов вернёт вложения за 1–3 месяца.
- **6–7 «Да»:** Операционка стоит вам серьёзных денег. Это требует системного подхода.

## Анализ разрыва

**Какую проблему создаёт лид-магнит:**
Человек заполняет чеклист и видит конкретную цифру потерь в месяц. Это создаёт острый когнитивный разрыв: «я теряю X рублей, и это уже подтверждено расчётом, а не ощущением».

**Что лид-магнит не решает:**
Он показывает масштаб проблемы, но не даёт готового решения для конкретного бизнеса. Человек знает «что» и «сколько», но не знает «как именно» применительно к своей ситуации.

**Как это закрывает оффер:**
Бесплатный 30-минутный разбор — это следующий естественный шаг: «теперь мы смотрим конкретно на ваш бизнес». Переход не выглядит как продажа — он выглядит как логичное продолжение начатой работы.
