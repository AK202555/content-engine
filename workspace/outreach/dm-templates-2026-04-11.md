## Ситуативные шаблоны

**Человек написал пост о боли, которую мы решаем:**
```
[ИМЯ], читал твой пост про [ТЕМА] — узнаваемая история.

Работаю именно с такими ситуациями. Если хочешь — могу поделиться тем, что обычно работает. Без питча, просто по теме.
```

**Человек задал вопрос в комментариях:**
```
[ИМЯ], ответил в комментариях, но напишу и сюда — чтобы не потерялось.

[ОТВЕТ НА ВОПРОС]

Если хочешь копнуть глубже применительно к вашей ситуации — пиши.
```

**Реакция + посещение профиля (холодное касание):**
```
[ИМЯ], привет! Видел, что смотрел профиль — если что-то зацепило, рад ответить на вопросы.

Занимаюсь AI-автоматизацией для МСБ. Если тема актуальна — напиши.
```

**После бесплатного разбора — фолоу-ап:**
```
[ИМЯ], спасибо за разговор сегодня.

Как и обещал — скину краткое резюме: [2–3 пункта из разговора + рекомендация].

Если решишь двигаться дальше — напиши, обсудим следующий шаг. Если нет — всё равно удачи с [КОНТЕКСТ].
```

## Правила использования

- Всегда адаптировать под конкретного человека — не копировать дословно
- Первое сообщение: ноль питча, ноль ссылок на услуги
- Каждое сообщение должно читаться за 30 секунд
- Если нет ответа после трёх касаний — остановиться
- Отвечать на ответы в течение нескольких часов, не дней
